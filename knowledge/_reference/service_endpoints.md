# Service Endpoints (DNS/URLs)

This list contains **only** endpoints an end user may need to access services.

## Identity / Microsoft 365
- Microsoft sign-in: https://login.microsoftonline.com
- Microsoft 365 portal: https://portal.office.com

## Email (Microsoft 365)
- Outlook on the web: https://outlook.office.com

## File storage (Microsoft 365)
- SharePoint tenant: https://acme.sharepoint.com
- OneDrive access: https://portal.office.com → OneDrive

## VPN (Cisco Secure Client / AnyConnect)
- VPN gateway: **vpn.acme.com**

## Salesforce
- ACME Salesforce friendly URL: **https://salesforce.acme.com**
- Salesforce sign-in (fallback): https://login.salesforce.com
